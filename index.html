Meus Projetos

Tela de Login - Top Style


Interface desenvolvida para a marca TopStyle Modas, com foco em usabilidade e identidade visual. O layout é dividido em duas seções: acesso rápido via email ou redes sociais, e um convite à criação de conta, com destaque para o logotipo e cores elegantes. O design equilibra simplicidade e sofisticação, oferecendo uma experiência clara e moderna ao usuário.

Tela de Portifólio - Windows XP


Este projeto é um portfólio pessoal desenvolvido com design inspirado na clássica interface do Windows XP. A proposta é unir nostalgia e criatividade, apresentando minhas informações e projetos como se fossem atalhos em uma área de trabalho retrô. Cada seção (Sobre Mim, Projetos, Contato, etc.) é representada por um ícone semelhante aos da época, criando uma navegação lúdica e intuitiva.

Tela de Login - Red Dead Redempition


Neste projeto, o design do login foi inspirado no universo de Red Dead Redemption, transportando o usuário para a ambientação do Velho Oeste, com uma interface que remete ao estilo rústico e detalhado do jogo. A tela de login exibe um papel antigo e amassado, com ícones e fontes que remetem ao clima do jogo, incluindo o famoso personagem John Marston.
